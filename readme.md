## Description

A LinkTree Backend (and Frontend) clone project created as a learning project. It uses Node, Express and Prisma.

## Installation

```bash
git clone https://github.com/marutina98/linktree-clone-backend
cd linktree-clone-backend
npm install
npm run dev
```
